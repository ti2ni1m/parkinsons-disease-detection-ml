# Parkinson’s Disease Detection using Machine Learning
## Overview

This project uses machine learning to detect Parkinson’s Disease based on voice measurements.

Implemented:

* Baseline model (DataFlair approach)
* Improved model with EDA and better preprocessing

## Technologies Used
* Python
* Zcikit-learn
* XGBoost
* Pandas, NumPy
* Matplotlib, Seaborn

## Workflow
1. Data preprocessing
2. Feature scaling
3. Model training (XGBoost)
4. Evaluation
## Results
Baseline Accuracy: 94%
Improved Accuracy: 95%

Both models achieved similar accuracy because XGBoost is not sensitive to feature scaling.

## Key Insights
* Feature scaling has minimal impact on tree-based models
* Voice features are strong indicators of Parkinson’s Disease
## Limitations
* Small dataset
* No real-world validation
